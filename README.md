# 会議室予約システム 

1つの画面内で、会議室の選定・予約情報の入力・登録された予約の確認までを行うwebアプリケーションです。

---

## クイックスタート

### 1. 依存パッケージのインストール
プロジェクトのルートディレクトリで以下のコマンドを実行してください。
```bash
npm install
# または pnpm を使用している場合
pnpm install
```
### フォルダ構造

```
src/
├── App.tsx            
└── kadai7/
    └── 8/
        ├── Dashboard.tsx    # 全体の状態（State）を統括する親コンポーネント
        ├── RoomList.tsx     # 利用可能な会議室をカード表示するコンポーネント
        ├── ReserveForm.tsx  # 予約入力・バリデーションを担当するコンポーネント
        └── type.tsx         # アプリケーション全体で共通利用する型定義ファイル
```
